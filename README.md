Dark, cinematic art gallery with fog, bubbles, glowing UI and placement-style auth.
Includes Gallery, Stories, Exhibitions, About, Contact, Login/Signup, and a role Dashboard with Favorites.

## Run
```bash
npm install
npm run dev    # http://localhost:5174
```

## Features
- React + Vite + Tailwind + Framer Motion
- Global fog/bubbles background + theme toggle + ambient sound toggle
- Gallery with mood filters and favorites (localStorage)
- Virtual lore: Stories page; Exhibitions page timeline cards
- Role-based dashboard (Visitor, Artist, Curator, Necromancer/Admin)
